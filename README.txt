Магазин на jquery


Реализовать подобие интернет-магазина.

В левой части страницы - перечень категорий.

При клике на категорию выводится список товаров этой категории.

Клик на товар - информацию о товаре.

В информации товара - кнопка “купить”

При клике на “купить” ниже появляется форма оформления заказа с следующими полями:

ФИО покупателя

Город (выбор из списка)

Склад Новой почты для отправки

Наложенный платеж или оплата банковской картой

Количество приобретаемой продукции

Комментарий к заказу

Реализовать проверку всех пользовательских данных - обязательные поля заполнены. В противном случае, выводить ошибку на страницу
